Array Problems

  1. printReverse()

    a function that takes an array as an argument and prints out the elements in reverse order.

  2. isUniform()

    a function which takes an array as an argument and returns true if all elements in the array are identical.

  3. sumArray()

    a function that accepts an array of numbers and returns the sum of all numbers in the array.

  4. max()

    a function that accepts an array of numbers and returns the max number in the array.
